System Architecture

  • AI Integration: Connects to Gemini 2.5 Flash via REST to securely generate unpredictable study intervals.
  • Secret Management: API keys are completely stripped from source code. GitHub Actions injects a Base64-encoded key during the CI/CD build step to bypass public repository secret scanning.
  • Data Persistence: Fully serverless data layer using Firebase Firestore to synchronize session arrays mathematically grouped by month.
  • Continuous Testing: The Python `pytest` suite strictly validates time-horizon boundaries and aggregation logic. Deployments to GitHub Pages are instantly aborted if any tests fail in the CI/CD pipeline.
